# Ant-pro

Ant-pro地址：https://procomponents.ant.design/components

## 1.本项目通过umi.js安装的 Ant-pro

​	优点：便利

`@umijs/max` 模板项目，更多功能参考 [Umi Max 简介](https://umijs.org/docs/max/introduce)

## 2.快速启动

1. 安装pnpm

```js
curl -fsSL https://get.pnpm.io/install.sh | sh -

$ pnpm -v
7.3.0
```

2. 项目环境安装

```
pnpm install
```



3.项目启动

```js
pnpm dev
```
